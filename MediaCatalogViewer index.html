Hybrid Engines

FFmpeg、QuickLook、Qtのハイブリッド構造を採用。形式に合わせて最適なエンジンを自動選択することで、多様なメディアファイルの確実なプレビュー表示を実現します。

Finder Tag Sync

macOS標準のタグ情報をリアルタイムに同期。Finderで設定したカラーラベルをそのままアプリ内で反映し、直感的なファイル仕分けと効率的なフォルダ整理をサポートします。

🎯

Precision Ops

確実なファイルハンドリングを重視。ドラッグ＆ドロップによる安全な移動、ゴミ箱への破棄、そしてmacOS標準の Cmd+Z によるUndo（取り消し）操作を完備しています。

🔍

Immersive View

没入感のあるシングルプレビュー機能を搭載。トラックパッドによる滑らかなズーム操作や自動スライドショーにより、膨大なメディアコレクションを快適に閲覧可能です。

Advanced Features

高速スキャンエンジン ＆ キャッシュ管理

独自のバックグラウンド走査システムにより、大量のファイルが存在するディレクトリでも軽快に動作します。生成されたサムネイルはローカルに保存され、2回目以降の閲覧はさらにスムーズになります。メモリ消費を抑えた設計により、長時間でも安定して動作します。

「不可視ファイル」の可視化と全領域アクセス

Finderの設定を変更することなく、通常は隠されているシステムファイルやライブラリ内のファイルを瞬時に可視化。通常のメディアファイルと同様にサムネイル化し、シームレスな整理・管理を可能にします。一般的なビューアでは手が届かない領域までを完全に掌握できる、プロフェッショナルのための独自機能です。 *システム保護領域や設定ファイルへの不用意なアクセス・操作は、OSの動作を致命的に損なうリスクを伴います。本機能の利用にあたっては、すべてユーザー自身の判断と責任において行ってください。

シームレスな「Undo」履歴管理

ファイル操作の履歴を適切に管理。誤ってファイルを移動した場合も、使い慣れた Cmd+Z ショートカットで元の場所へ戻せます。物理的なファイル操作に伴う不安を解消し、安心して整理作業に集中できます。

外部・ネットワークドライブのサポート

Mac本体だけでなく、外付けHDD、SSD、NASなどのネットワークボリュームも標準サポート。サイドバーからワンクリックでボリュームを切り替え可能。ネットワーク上のファイルもメタデータ管理によりストレスなく閲覧できます。
