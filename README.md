# Media Catalog Viewer
### Ultimate High-Speed Media & File Organizer for Mac

ディスクの底に埋もれた膨大なファイルを一瞬で見つけ出し、意のままに整理する。
画像・動画・ドキュメントを視覚的に一望し、驚きのスピードで仕分けるプロ仕様のMac専用カタログビューアです。

---

## 🚀 主な特徴 (Core Features)

### 🎥 10% Duration Snapshot (業界初・3段階解析)
動画の冒頭が暗転していても、本質を見逃しません。総再生時間の10%地点をピンポイントで特定して描画する革新的なエンジンを搭載。

### 🏷️ Finder Tag Sync (C-API統合)
macOS標準のタグ情報をリアルタイム同期。Finderで設定したカラーラベルをそのままアプリ内で反映し、シームレスなファイル管理を実現します。

### ⚙️ Hybrid Preview Engines
FFmpeg、QuickLook、Qtのハイブリッド構造を採用。形式に合わせて最適なエンジンを自動選択し、あらゆるメディアのプレビューを可能にします。

### 🎯 Precision Ops & Undo
Appleシリコン(M1/M2/M3)に最適化された爆速のレスポンス。ドラッグ＆ドロップ操作はもちろん、誤操作を防ぐ **Cmd+Z (Undo)** にも完全対応。
